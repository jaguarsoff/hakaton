### Как запустить?

_Чтобы запустить бэкенд собиратите проект react и запустите через php в корне_

1. Клонировать
   ```sh
   git clone https://github.com/jaguarsoff/hakaton
   ```
2. Установка
   ```sh
   npm install
   ```
3. 
   ```sh
   npm run start
   ```
